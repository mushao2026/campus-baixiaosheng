## 🚀 使用方法

### 方式一：本地启动（推荐）
```bash
# 1. 解压项目文件
# 2. 进入项目目录
cd campus-assistant

# 3. 安装依赖
npm install

# 4. 启动前端（终端1）
npm run dev

# 5. 启动后端（终端2）
npm run server

# 6. 打开浏览器访问 http://localhost:3000
```

### 方式二：直接使用（静态版）
如果不想启动服务器，可以直接打开 `index.html`（但需要先修改Chat组件，去掉API调用）
