# 知识库索引 (Knowledge Base Index)

本目录包含大学生活智能体的全部知识模块，面向大一新生提供全方位指导。

## 📚 知识模块列表

### 1️⃣ 选课攻略 (01-选课攻略.md)
**适用场景**：
- 新手入学，不了解选课流程
- 想要抢到热门课程
- 需要解读培养方案

**核心内容**：
- 选课轮次（预选、正选、补退选）
- 必修/选修区别
- 抢课技巧
- 退课规则

---

### 2️⃣ 奖学金与资助 (02-奖学金与资助.md)
**适用场景**：
- 想申请奖学金但不知道条件
- 家庭经济困难，需要资助
- 想了解综测与GPA的关系

**核心内容**：
- 国家奖学金、励志奖学金、助学金
- 校内奖学金、社会奖学金
- 申请条件和材料准备
- 评选时间和流程

---

### 3️⃣ 学分与GPA (03-学分与GPA.md)
**适用场景**：
- 不了解学分体系和GPA计算
- 想要提升GPA排名
- 关心保研、出国对成绩的要求

**核心内容**：
- 学分类型和要求
- GPA计算方法和提升策略
- 挂科补救措施
- 成绩认证（出国）

---

### 4️⃣ 社团与实践 (04-社团与实践.md)
**适用场景**：
- 想参加社团但不知道选哪个
- 担心社团活动影响学业
- 想要提升综合素质

**核心内容**：
- 社团类型如何选择
- 实践活动种类（实习、竞赛、交流）
- 学业与社团的平衡
- 时间管理技巧

---

### 5️⃣ 竞赛指南 (05-竞赛指南.md)
**适用场景**：
- 想参加竞赛但不了解种类
- 需要组队和备赛建议
- 想评估竞赛含金量

**核心内容**：
- 竞赛分类（按学科、难度）
- 2026年重点竞赛时间表
- 备赛时间线和组队策略
- 竞赛经历在简历中的呈现

---

### 6️⃣ 大学排名 (06-大学排名.md)
**适用场景**：
- 高中生择校参考
- 大学生规划深造（保研/出国）
- 想理性看待大学排名

**核心内容**：
- 主流排名体系（软科、QS、THE等）
- 排名指标解读和局限性
- 如何利用排名择校/规划
- 超越排名的选择标准

---

### 7️⃣ 学习方法 (07-学习方法.md)
**适用场景**：
- 不适应大学学习方式
- 想要提升学习效率
- 需要应对考试和论文

**核心内容**：
- 大学学习特点（与高中区别）
- 高效学习法（费曼、番茄、康奈尔等）
- 时间管理工具和方法
- 各科学习策略（理工/文科/商科）

## 🎯 使用建议

1. **按需求查阅**：根据自己的问题，直接查看对应模块
2. **交叉参考**：某些问题可能涉及多个模块（如"竞赛"可能涉及"学习方法"和"社团实践"）
3. **及时更新**：政策每年可能变化，建议关注学校官网最新通知
4. **结合实际**：文章内容为通用建议，具体以你所在学校的规定为准
