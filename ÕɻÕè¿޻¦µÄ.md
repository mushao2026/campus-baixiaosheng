# 大学生活智能体 - 快速启动指南

## 📦 项目已完整创建！

本项目包含：
- ✅ 7大知识模块（选课、奖学金、GPA、社团、竞赛、排名、学习方法）
- ✅ 3个子智能体（选课顾问、竞赛规划师、奖学金助手）
- ✅ React + TypeScript 前端界面
- ✅ Mock后端服务
- ✅ 完整项目文档

---

## 🚀 启动步骤

### 1️⃣ 安装依赖

进入项目目录，运行：

```bash
cd campus-assistant
npm install
```

### 2️⃣ 启动前端（终端1）

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动

### 3️⃣ 启动后端Mock服务（终端2）

```bash
npm run server
```

后端将在 `http://localhost:3001` 启动

### 4️⃣ 开始使用

打开浏览器访问 `http://localhost:3000`，即可开始对话！

## ❓ 常见问题

### Q: npm install 很慢怎么办？
A: 使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口被占用怎么办？
A: 修改 `vite.config.ts` 中的端口号，或关闭占用端口的程序。

### Q: 如何修改回答内容？
A: 直接编辑 `knowledge/` 目录下的对应Markdown文件，然后重启后端服务。
